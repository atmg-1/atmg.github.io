## 🎨 eDEX-UI 配色方案

### 主要颜色
```css
--cyber-primary: #00ffea      /* 青色 - 主色调 */
--cyber-secondary: #ff6b00    /* 橙色 - 强调色 */
--cyber-accent: #00ff9f       /* 绿色 - 点缀色 */
--cyber-alert: #ff2a00        /* 红色 - 警告色 */
```

### 应用场景
- **青色**: 标题、边框、按钮、进度条
- **橙色**: 悬停效果、页脚边框
- **绿色**: 成功状态、特殊元素
- **红色**: 警告、错误提示

## 💡 自定义效果开关

如果想关闭某些效果，编辑 `source/js/cyberpunk-effects.js`:

```javascript
const CONFIG = {
  enableGlitch: false,        // 关闭故障艺术
  enableGridBackground: false, // 关闭网格背景
  enableTimeControl: false,    // 关闭时间线控制
};
```

然后重新推送：
```bash
git add .
git commit -m "调整效果配置"
git push origin main
```
